# Voltage Divider Analysis with Uncertainty Propagation

This module analyzes a voltage divider circuit considering component tolerances. Uncertainty propagation is used to estimate the output voltage range and compare it with experimental measurements.

# Methodology 
1. Define the circuit parameters (input voltage and resistances with tolerance).
2. Compute the nominal output voltage using the voltage divider equation.
3. Apply uncertainty propagation using partial derivatives to estimate output variation.
4. Determine the theoretical tolerance interval of the output voltage.
5. Build the circuit and measure the output voltage experimentally.
6. Compare experimental results with the theoretical interval to validate the model.
