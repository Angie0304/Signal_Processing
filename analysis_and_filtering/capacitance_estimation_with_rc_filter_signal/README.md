# Capacitance Estimation Using an RC Filter and Signal Analysis

This module presents the estimation of capacitance in an RC low-pass filter by analyzing its output signal. The capacitance is calculated using the theoretical exponential model and experimental measurements obtained from an oscilloscope.

## Methodology 

1. An RC low-pass filter was implemented using a known resistance and capacitor.  
2. A square input signal was applied to the circuit.  
3. Input and output signals were measured using an oscilloscope.  
4. The capacitor voltage was analyzed during the charging process.  
5. Capacitance was estimated using the exponential model of the RC circuit.  
6. The estimated value was compared with the nominal capacitor value.
