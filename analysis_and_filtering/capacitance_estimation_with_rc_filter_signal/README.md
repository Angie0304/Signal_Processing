# Capacitance Estimation Using an RC Filter and Signal Analysis

This project presents the estimation of capacitance in an RC low-pass filter by analyzing its output signal. The capacitance is calculated using the theoretical exponential model and experimental measurements obtained from an oscilloscope.

## Methodology 

- An RC low-pass filter was implemented and excited with a square input signal.  
- Input and output signals were measured using an oscilloscope.  
- The capacitor response was analyzed during the charging process.  
- Capacitance was estimated using the exponential model.  
- The result was compared with the nominal value.
